# Democracy Index Analysis

## Overview
Analyzed global democracy index data to identify patterns, trends, and differences across countries and regions.

## Tools Used
- Python (Pandas, NumPy)
- Data visualization (Matplotlib / Seaborn)

## Key Skills Demonstrated
- Data cleaning and preparation
- Exploratory data analysis
- Insight-driven visualization

## Outcome
Identified meaningful trends and comparisons in democratic indicators across countries to support data-driven insights.
